This repository demonstrates Infrastructure as Code (IaC), cloud provisioning, configuration management, and automation practices.

- Terraform
- AWS
- Microsoft Azure
- Ansible
- Python
- YAML

## Areas Covered

- Infrastructure Provisioning
- Cloud Networking
- Compute Automation
- Configuration Management
- Automation Scripts
